# Rock Paper Scissors Game

A simple Rock Paper Scissors game built with vanilla JavaScript, HTML, and CSS.

## Features

- Play against the computer with Rock, Paper, or Scissors choices.
- Real-time display of round results and scores.
- Lightweight, fast, and runs entirely in the browser.

## Tech Stack

- JavaScript – Core game logic. 
- HTML – Page structure and layout. 
- CSS – Basic styling and visual presentation.


## Getting Started

1. Clone the repository:
2. Open the project folder.
3. Open `index.html` in your preferred web browser.

The game will load in the browser and you can start playing immediately.

## How to Play

- Click on one of the options: Rock, Paper, or Scissors.
- The computer will randomly choose its option.
- The result of the round (win, lose, or draw) is displayed along with updated scores.

## Project Structure

- `index.html` – Main game page and layout.
- `style.css` – Styling for the game UI.
- `script.js` – Game logic and interactivity.
